# 分解质因数
```
void check(int m) {
    for (int i = 2; i <= m / i; i++) {
//这里不需要再判断素数了，证明如下：
/*
假设i是一个合数，那么必然可以分解成多个质因数相乘的形式，
且这些质因数全部小于i，那么必然已经在之前的结果里循环过了
即已经把m给除去了这些质因数。例如4 = 2 * 2, 那么m已经
进行了两次除2的操作，也即m /= 2, m /= 2，那样就不会使得
条件“m % i == 0 ”成立，自然也就不会有后面的结果了。
*/
        if (m % i == 0) {
            int num = 0;
            while (m % i == 0) {
                m /= i;
                num++;
            } cout << i << ' ' << num << endl;
        } 
    } 
    
    if (m > 1) cout << m << ' ' << 1 << endl;
    cout << endl;
}
```
