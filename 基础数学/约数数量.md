# 约数个数
- 要求：给定若干个取值范围在int内的正整数，求它们的乘积值的约数个数。
- 定理：设正整数P = (x1 ^ y1)(x2 ^ y2)...(xk ^ yk)，则其约数的数量为(y1 + 1)(y2 + 1)...(yk + 1).

```
int n, m;

unordered_map<int, int> um;

const int mod = 1e9 + 7;

int main() {
    int n; cin >> n;
    while (n--) {
        cin >> m;
        for (int i = 2; i <= m / i; i++) {
            while (m % i == 0) {
                um[i] = (um[i] + 1) % mod;
                m /= i;
            }
            
        } if (m > 1) um[m] = (um[m] + 1) % mod;
    }
    unsigned long long ans = 1;
    for (auto u: um) {
        ans = (ans * (u.second + 1)) % mod;
    } cout << ans;
}
```
